# Connections Game

A web-based word connection game similar to NYT Connections.

## How to Play

- 16 words are displayed in a grid.
- Find 4 groups of 4 words that belong together.
- Select 4 words and submit to check if they form a correct group.
- Continue until all groups are found.

## Running Locally

1. Ensure Python 3 is installed.
2. Run `python3 -m http.server 8000` in the project directory.
3. Open `http://localhost:8000` in your browser.

## Features

- Multiple puzzles
- Simple UI
